# Personal Portfolio Website

This repository contains the source code for my personal portfolio
website. The website presents my background, projects, and resume as a
Computer Science graduate with a focus on frontend and web development.

## About the Project

This portfolio was built using **React** and modern web technologies. It
provides information about my education, professional experience, and
projects. The website also supports **English and German language
versions**.

The goal of this project is to present my work and technical skills in a
clear and accessible way for potential employers and collaborators.

## Features

-   Responsive portfolio website
-   English and German language support
-   Projects overview
-   Resume section
-   Contact section
-   Clean and simple UI

## Technologies Used

-   React
-   JavaScript (ES6+)
-   HTML5
-   CSS3
-   Vite
-   Git & GitHub
